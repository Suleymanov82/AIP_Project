# My App

## Run The Project

If Node.js was installed through `nvm`, load it in the current terminal:

```bash
export NVM_DIR="$HOME/.nvm"
. "$NVM_DIR/nvm.sh"
```

Then start the development server:

```bash
cd /Users/rustem/Downloads/my-app
npm run dev
```

You can also use:

```bash
npm start
```

The app will be available at:

- `http://127.0.0.1:5173/`

## Available Scripts

- `npm run dev` - start the Vite development server
- `npm start` - same as `npm run dev`
- `npm run build` - build the app for production
- `npm run preview` - preview the production build on `http://127.0.0.1:4173/`
- `npm run lint` - run ESLint
