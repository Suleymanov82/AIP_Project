# АвтоДеталь — Маркетплейс автозапчастей

Учебный проект — маркетплейс для покупки и продажи автозапчастей. Разработан в рамках курса АИП. Пользователи могут просматривать объявления, фильтровать по категориям и добавлять собственные.

## Технологический стек

- **Node.js** + **Express.js** — серверная часть
- **MongoDB** + **Mongoose** — база данных
- **EJS** — шаблонизатор
- **Bootstrap 3** — UI
- **Font Awesome 6** — иконки
- **express-session** + **connect-mongo** — сессии и авторизация

## Функциональность

- Каталог запчастей с фильтрацией по категории, марке, модели и состоянию
- 8 категорий: Двигатель, Трансмиссия, Тормозная система, Подвеска, Электрика, Кузов, Охлаждение, Топливная система
- Регистрация и вход с хранением сессий в MongoDB
- Добавление объявлений авторизованными пользователями
- Детальная страница товара с характеристиками и информацией о продавце
- Поиск по названию запчасти
- Адаптивный дизайн

## Установка и запуск

Требуется Node.js и MongoDB.

```bash
# Установить зависимости
npm install

# Заполнить базу данных тестовыми данными
node createDB.js

# Запустить сервер
npm start
```

Сайт будет доступен по адресу: [http://localhost:3000](http://localhost:3000)
